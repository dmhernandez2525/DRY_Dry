# dry

> Reusable react components

[![NPM](https://img.shields.io/npm/v/dry.svg)](https://www.npmjs.com/package/dry) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save dry
```

## Usage

```jsx
import React, { Component } from "react";

import MyComponent from "dry";
import "dry/dist/index.css";

class Example extends Component {
  render() {
    return <MyComponent />;
  }
}
```
