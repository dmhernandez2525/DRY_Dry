# React Standards

### Imports

imports should have this structure:

```jsx
// Outside packages
import React from "react";

// Internal components
import Header from "../Header";

// Utilities
import * as ApiCalls from "../../utils/apiCalls.js";

// Styling
import "./Home.scss";

// Everything else
import README from "./README";
```

### NOTE

Each set of imports should be separated by a space
